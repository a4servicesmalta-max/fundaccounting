# Quick Start

Six steps and you're running.

1. **Install Node.js** (version 20 or newer) from <https://nodejs.org/> — download the "LTS" version and run the installer.
2. **Open a terminal in this folder.** On Windows: open this folder, then type `cmd` in the address bar and press Enter.
3. **Install the app** — type this and press Enter:
   ```
   npm install
   ```
4. **Add your API key** — make a copy of `.env.example` and name the copy `.env`, then open `.env` and paste your Anthropic API key (get one at <https://console.anthropic.com/>) after `ANTHROPIC_API_KEY=`.
5. **Start the app** — type this and press Enter:
   ```
   npm start
   ```
6. **Open it in your browser** — go to <http://localhost:4350>.

That's it. Drop in your documents and review the entries it prepares.
